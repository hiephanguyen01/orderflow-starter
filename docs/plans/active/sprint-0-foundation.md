# Sprint 0 — Foundation

## Goal

Create a runnable monorepo and stable engineering conventions before implementing business modules.

## Completed

- pnpm workspace and Turborepo configuration;
- NestJS API and background worker;
- Next.js storefront and administration applications;
- HeroUI v3 and Tailwind CSS v4;
- Vietnamese and English next-intl routing;
- PostgreSQL, Redis and MinIO local infrastructure;
- Prisma v7 configuration;
- shared UI, API client, contracts and TypeScript configuration;
- health endpoint and Swagger bootstrap;
- AGENTS.md, skills and engineering documentation.

## Exit criteria

- formatting passes;
- lint passes;
- TypeScript checks pass;
- all workspaces produce a production build;
- local infrastructure can be started with Docker Compose.
