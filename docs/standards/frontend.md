# Frontend Standards

- Use Next.js App Router and TypeScript strict mode.
- Use HeroUI before creating custom interactive primitives.
- Use TanStack Query for server state.
- Use React Hook Form and Zod for forms.
- Use the shared Axios client; components do not call Axios directly.
- Use next-intl for static text.
- Dynamic business names such as runtime roles are displayed from API data.
- Always implement loading, empty, error and success states.
- Hidden controls never replace backend authorization.
